# HibernateBookAuthor in Spring
This exercise involved the creation of a library with books and authors.

The books could be written by many authors and the authors could write different books.

# Challenges
Get used to Hibernate / JPA and their annotations.
Use a database and create tables directly from Spring.

# What I'Ve Learned:
Learned the basics to use JPA Repository inside the Services.

Learned how to connect Spring with a database and use Hibernate to create Tables (ManyToMany Table).
